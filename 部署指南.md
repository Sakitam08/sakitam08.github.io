## 🚀 部署方法（推荐按顺序尝试）

### 方法一：GitHub Pages（免费，推荐）

**优点：** 完全免费、稳定、支持自定义域名、GitHub开发者友好

**步骤：**

1. **创建GitHub账号**
   - 访问 https://github.com 注册账号

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名称：`your-username.github.io`（替换your-username为你的用户名）
   - 设置为 Public
   - 点击 "Create repository"

3. **上传文件**
   - 在仓库页面点击 "uploading an existing file"
   - 将 `index.html`、`style.css`、`script.js` 三个文件拖拽上传
   - 点击 "Commit changes"

4. **访问网站**
   - 等待1-2分钟
   - 访问：`https://your-username.github.io`

5. **（可选）自定义域名**
   - 在仓库设置 → Pages → Custom domain
   - 输入你的域名（如：www.yourname.com）
   - 在域名DNS设置中添加CNAME记录指向 `your-username.github.io`

---

### 方法二：Vercel（免费，速度快）

**优点：** 部署速度快、自动HTTPS、支持自定义域名

**步骤：**

1. **注册Vercel**
   - 访问 https://vercel.com 注册（可用GitHub账号登录）

2. **导入项目**
   - 点击 "New Project"
   - 如果已上传到GitHub，直接导入GitHub仓库
   - 或选择 "Upload" 直接上传文件夹

3. **部署**
   - 点击 "Deploy"
   - 等待部署完成（通常1-2分钟）

4. **访问网站**
   - Vercel会提供一个 `.vercel.app` 域名
   - 可在设置中添加自定义域名

---

### 方法三：Netlify（免费，拖拽部署）

**优点：** 最简单、支持拖拽上传、自动HTTPS

**步骤：**

1. **注册Netlify**
   - 访问 https://netlify.com 注册

2. **拖拽部署**
   - 登录后，将包含网站文件的文件夹直接拖拽到页面中
   - 等待上传和部署完成

3. **访问网站**
   - Netlify会提供一个随机域名
   - 可在 "Site settings" → "Change site name" 修改域名

---

### 方法四：Gitee Pages（国内访问快）

**优点：** 国内访问速度快、中文界面

**步骤：**

1. **注册Gitee**
   - 访问 https://gitee.com 注册账号

2. **创建仓库**
   - 创建新仓库，设置为公开

3. **上传文件**
   - 上传 `index.html`、`style.css`、`script.js`

4. **开启Gitee Pages**
   - 仓库页面 → "服务" → "Gitee Pages"
   - 点击 "启动"
   - 选择部署分支（通常为master）
   - 点击 "更新"

5. **访问网站**
   - 等待几分钟后访问提供的域名

## 🆘 常见问题

**Q: 网站打不开？**
A: 检查文件名是否正确（index.html不是index.htm），等待几分钟让DNS生效

**Q: 样式不显示？**
A: 确保style.css和index.html在同一目录

**Q: 如何添加更多页面？**
A: 创建新的HTML文件，在导航栏添加链接

**Q: 如何添加在线支付？**
A: 可以集成支付宝、微信支付的收款二维码
